# ingest-pyspark

**ingest-pyspark** is an ingestion tool written in PySpark and designed to extract data from source systems and persist it into Delta Tables. 

It is intended for [Big Data](big-data) datasets and forms the **Extract and Load** portion of an ELT pipeline, following a [medallion architecture](medallion-architecture).
